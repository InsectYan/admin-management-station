# test_category_minor_scheme · 子类默认 TS/VS 映射

> 由 `scheme-map.json` + 维度/大类默认生成，供 `resolve-item-schemes.js` 在 prefix 规则未命中时兜底。

| 字段 | 说明 |
|------|------|
| category_minor_id | PK，关联 test_category_minor |
| item_prefix | 对应 test_item_id 前缀（如 C1-MACRO-） |
| scheme_primary_id / scheme_secondary_id | 主/辅测试方案 TS |
| validation_primary_id / validation_secondary_id | 主/辅验证标准 VS |
| mapping_source | scheme-map.json 或 dimension/major-default |
