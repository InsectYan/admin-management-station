# test_item_detail · 测试核心细节主表

> 383 条可执行测试项，汇聚分类、步骤、TS/VS、六站、自动化与 PRD/架构引用。

共 **383** 条。

## 表用途

Fitness 测试体系核心主数据；列表/详情/导出/计划批量加入均读写本表。关联表扩展多对多 PRD/架构/风险关系。

## 字段说明

| 字段 | 类型 | 约束 | 中文名 | 说明 |
|------|------|------|--------|------|
| `item_id` | VARCHAR(64) | PK | 用例编码 | 主键，如 C1-MACRO-001 |
| `dimension_id` | CHAR(1) | NOT NULL FK→test_dimension | 维度 | FK → test_dimension |
| `category_major_id` | VARCHAR(32) | NOT NULL FK→test_category_major | 大类 | FK → test_category_major |
| `category_minor_id` | VARCHAR(64) | NOT NULL FK→test_category_minor | 子类 | FK → test_category_minor |
| `sub_class` | VARCHAR(128) | — | 子类标签 | md 子类列或 GENERAL |
| `item_name` | VARCHAR(512) | NOT NULL | 用例名称 | 可读标题，含编码与子类摘要 |
| `detail_summary` | TEXT | NOT NULL | 核心细节 | 测什么：核心细节一句话 |
| `expected_observation` | TEXT | — | 期望观测 | 期望观测/断言摘要 |
| `test_input_example` | TEXT | — | 输入示例 | 示例输入（意图句、载荷等） |
| `preconditions` | JSONB | NOT NULL | 前置条件 | JSONB 前置条件数组 |
| `test_steps` | JSONB | NOT NULL | 测试步骤 | JSONB 执行步骤数组 |

| `config_env_id` | VARCHAR(64) | FK→config_env_enum | 配置项 | FK → config_env_enum |
| `station_id` | VARCHAR(8) | NOT NULL FK→test_station_enum | 六站 | FK → test_station_enum，默认 NONE |
| `role_scope_id` | VARCHAR(16) | NOT NULL FK→test_role_enum | 三端 | FK → test_role_enum，默认 ALL |
| `endpoint_path` | VARCHAR(256) | — | 接口路径 | HTTP 路径，如 /api/chat/turns/submit |
| `http_method` | VARCHAR(8) | — | HTTP方法 | GET | POST 等 |
| `http_status_expected` | SMALLINT | — | 期望状态码 | 期望 HTTP 状态码 |
| `scheme_primary_id` | VARCHAR(16) | FK→test_scheme_enum | 主方案 | FK → test_scheme_enum 主 TS |
| `scheme_secondary_id` | VARCHAR(16) | FK→test_scheme_enum | 辅方案 | FK → test_scheme_enum 辅 TS |
| `validation_primary_id` | VARCHAR(20) | FK→test_validation_enum | 主验证 | FK → test_validation_enum 主 VS |
| `validation_secondary_id` | VARCHAR(20) | FK→test_validation_enum | 辅验证 | FK → test_validation_enum 辅 VS |
| `sample_execution_note` | TEXT | — | 执行说明 | 样本/重复/链路执行说明 |
| `scheme_mapping_source` | VARCHAR(128) | — | 方案映射来源 | 方案来源：scheme-map.json 或 *-方案映射.md |
| `is_risk_flag` | BOOLEAN | NOT NULL | 风险项 | 是否高风险清单项 |
| `is_observability_audit` | BOOLEAN | NOT NULL | 可观测稽核 | 是否可观测稽核项 |
| `is_p0_blocker` | BOOLEAN | NOT NULL | P0阻塞 | 是否 P0 发版阻塞项 |
| `failure_symptom` | TEXT | — | 失败症状 | F5 症状→测试 描述 |

| `tags` | JSONB | NOT NULL | 标签 | JSONB 标签数组 |

| `source_doc` | VARCHAR(128) | NOT NULL | 来源文档 | 来源 md 文件名 |

| `is_active` | BOOLEAN | NOT NULL | 启用 | 是否启用（软删标记） |
| `created_at` | TIMESTAMPTZ | NOT NULL | 创建时间 | 创建时间 |
| `updated_at` | TIMESTAMPTZ | NOT NULL | 更新时间 | 更新时间 |
